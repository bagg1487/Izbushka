# Избушка - веб-сервер управления роботом


## О проекте

Веб-сервер на FastAPI для удалённого управления роботом через браузер. Проект объединяет в себе шину команд (TCP-сокет) для связи с роботом, RTSP-видеопоток с камеры и веб-интерфейс. Управление возможно как с ПК (клавиатура/мышь), так и с мобильных устройств (сенсорное управление).

**Возможности:**
- Авторизация пользователей (JWT-токены, роль admin)
- Управление движением робота (вперёд/назад/влево/вправо/стоп)
- Прямая трансляция RTSP-видеопотока через WebSocket (base64 JPEG)
- Калибровка сервоприводов (4 конечности, 5 сервоприводов A–E)
- Команды: помахать, музыка вкл/выкл, радио вкл/выкл, меню
- Информационные команды: о коде, плате, питании, нейросетях, настройках, системе
- Статус подключения к роботу в реальном времени
- Асинхронная обработка WebSocket-соединений для видео

## Технологический стек

| Компонент       | Технология                |
|-----------------|---------------------------|
| Язык            | Python 3.12+              |
| Веб-фреймворк   | FastAPI + Uvicorn         |
| Авторизация     | JWT (python-jose), bcrypt |
| Видеопоток      | OpenCV, WebSocket, base64 |
| Стили           | HTML5/CSS3, Font Awesome 6 |
| Сеть            | TCP-сокеты (порт 5001), RTSP (порт 8554) |
| Асинхронность   | asyncio, threading        |

## Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Настройка переменных окружения

Создайте файл `.env`:

```env
SECRET_KEY=your-super-secret-key-change-it
ROBOT_IP=192.168.15.2
ROBOT_COMMAND_PORT=5001
RTSP_URL=rtsp://192.168.15.2:8554/stream
```

### 3. Запуск сервера

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

### 4. Доступ к интерфейсу

Откройте в браузере: `http://localhost:8000`

## Возможности интерфейса

### Главная страница (index.html)
- Форма авторизации (JWT)
- Три кнопки:
  - **Управление** → Панель движения
  - **Видео** → Полноэкранный видеопоток
  - **Вести** → Информационные команды

### Панель управления (control.html)

| Элемент                | Действие                                   |
|------------------------|---------------------------------------------|
| Кнопки со стрелками    | Вперёд/Назад/Влево/Вправо                  |
| Кнопка Stop            | Остановка движения                         |
| Кнопка «Руки»          | Команда `CMD_HANDS_UP`                     |
| Кнопка «Музыка»        | `CMD_MUSIC_PLAY`                           |
| Клавиши W/A/S/D        | Аналог кнопок движения                     |
| Пробел                 | Остановка                                  |                      |
| Видео          | RTSP-поток через WebSocket (30 fps)        |
| Статус подключения     | Обновляется каждые 2 секунды               |

### Страница информации (info.html)

| Раздел         | Команды                                              |
|----------------|------------------------------------------------------|
| О себе         | О коде, О плате, О питании, О нейросетях, О настройках, О системе |
| Действия       | Помахать, Музыка вкл/выкл, Радио вкл/выкл, Меню     |

Каждая кнопка отправляет POST-запрос на соответствующий роут.

## API Endpoints

### Авторизация

| Метод | Эндпоинт          | Описание                    |
|-------|--------------------|-----------------------------|
| POST  | `/auth/login`      | Получить JWT-токен          |
| POST  | `/auth/logout`     | Выход (инвалидация на клиенте) |
| GET   | `/auth/me`         | Информация о текущем пользователе |

### Команды движения (требуют JWT)

| Метод | Эндпоинт                  | Параметры                |
|-------|----------------------------|---------------------------|
| POST  | `/commands/move/{direction}` | `direction`: forward, backward, left, right, stop |
| POST  | `/commands/hands`          | -                         |
| POST  | `/commands/music/play`     | -                         |
| POST  | `/commands/music/stop`     | -                         |
| POST  | `/commands/radio/play`     | -                         |
| POST  | `/commands/radio/stop`     | -                         |
| POST  | `/commands/menu`           | -                         |
| POST  | `/commands/about/{num}`    | `num`: 0–5                |

### Калибровка

| Метод | Эндпоинт                  | Описание                          |
|-------|----------------------------|------------------------------------|
| GET   | `/calibration/data`        | Получить текущие углы (4×5)       |
| POST  | `/calibration/set`         | Установить угол для сервопривода  |
| POST  | `/calibration/mode/{state}` | Включить/выключить режим калибровки |

### Пользователи

| Метод | Эндпоинт          | Описание                          |
|-------|--------------------|------------------------------------|
| POST  | `/users/add`       | Начать добавление пользователя   |
| POST  | `/users/photo`     | Сохранить фото с именем           |

### WebSocket

| Путь               | Описание                                      |
|--------------------|------------------------------------------------|
| `/ws/video`        | WebSocket-соединение для видеопотока (base64 JPEG, 30 fps) |

## Формат команд для робота

Все команды отправляются через TCP-сокет (порт 5001):

```
CMD_MOVE_FORWARD#50\n
CMD_CALIBRATION#L#A#90\n
CMD_TAKE_PHOTO#Alex\n
```
