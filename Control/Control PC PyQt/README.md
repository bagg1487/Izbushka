# Управление с помощью PyQt - клиент для управления роботом

## О проекте

Desktop-приложение на PyQt5 для управления роботом через TCP/IP. Позволяет подключаться к серверу робота, управлять движением, получать видеопоток, настраивать сервоприводы, распознавать пользователей, управлять музыкой и радио.

**Возможности:**
- Подключение к серверу робота по IP-адресу (порты 8001 — видео, 5001 — команды)
- Управление движением (вперёд/назад/влево/вправо) 
- Управление руками, головой (вверх/вниз)
- Голосовое управление (музыка, радио) через отправку команд на сервер
- Калибровка сервоприводов (левая/правая нога, левая/правая рука, 5 сервоприводов A–E)
- Добавление нового пользователя по фото с именем
- Получение видеопотока с сервера в реальном времени
- Получение данных с ультразвукового датчика (расстояние в cm)
- Поддержка клавиатуры (WASD, цифры 1–4, M/N, R/T, H, V, ESC) и геймпада (pygame)
- Отправка информационных команд (о себе, коде, платах, нейросетях, питании)
- Сохранение последнего подключённого IP в файл `IP.txt`


## Инструкция по подключению к роботу

1) В поле ввода указать IP-адрес сервера робота
(Порты сервера должны быть открыты: 8001 — видео, 5001 — команды)
2) Нажать кнопку Connect
3) При успешном подключении:
   - Появится видеопоток
   - Станут активны кнопки управления
   - Автоматически отправится команда калибровки CMD_CALIBRATION_ALL

## Инструкция по управлению

### Клавиатура

| Клавиша | Действие |
|---------|----------|
| **Движение** || 
| `W` | Вперёд |
| `S` | Назад |
| `A` | Влево |
| `D` | Вправо |
| `Пробел` | Стоп |
| **Действия** ||
| `1` | Левая рука |
| `2` | Правая рука |
| `3` | Голова вверх |
| `4` | Голова вниз |
| `H` | Помахать |
| **Медиа** ||
| `M` | Вкл музыку |
| `N` | Выкл музыку |
| `R` | Вкл радио |
| `T` | Выкл радио |

### Геймпад

| Элемент | Действие |
|---------|----------|
| **Движение** ||
| Левый стик (Y) | Вперёд / Назад |
| Левый стик (X) | Влево / Вправо |
| Правый стик (Y) | Голова вверх / вниз |
| **Действия** ||
| Кнопка `A` | Левая рука |
| Кнопка `B` | Правая рука |
| `Back` | Помахать |
| **Медиа** ||
| Кнопка `X` | Вкл музыку |
| Кнопка `Y` | Выкл музыку |
| `LB` | Вкл радио |
| `RB` | Выкл радио |
| **Прочее** ||
| `Start` | Меню |

## Калибровка

1) Нажать Калибровка

2) Выбрать ногу/руку (левая/правая нога, левая/правая рука)

3) Кнопками < и > регулировать углы сервоприводов A–E

4) Нажать Сохранить 

## Добавление пользователя

1) Нажать Добавить пользователя

2) Ввести имя

3) Навести камеру робота на лицо

4) Нажать Take Photo — сервер сохранит фото пользователя
