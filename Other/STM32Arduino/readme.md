# STM32F4 Программа для робота

## Что делает:

Управляет:

- **Джойстиком** (аналоговые оси X/Y + 4 кнопки)
- **Сервоприводами** (ноги и руки робота через PCA9685)
- **Моторами** (L298N драйвер)
- **Ультразвуковым датчиком** (HC-SR04)
- **Гироскопом/акселерометром** (MPU6050)

## Периферия

| Компонент | Назначение |
|-----------|------------|
| STM32F4 | Управляющий микроконтроллер |
| PCA9685 (x2) | 16-канальные PWM-драйверы для сервоприводов |
| L298N | Драйвер двигателей |
| HC-SR04 | Ультразвуковой дальномер |
| MPU6050 | Гироскоп + акселерометр |
| Аналоговый джойстик | Управление с 4 кнопками |
