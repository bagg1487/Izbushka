## Комплектующие / компоненты

- Плата Arduino
- TFT LCD Дисплей
- Датчик DHT11

## Необходимые библиотеки 

- LCDWIKI_GUI
- LCDWIKI_KBV - графика и управление дисплеем TFT LCD
- DHT sensor library - для работы с датчиком DHT11
