# Избушка — прошивка для робота 

---

## О проекте

Прошивка для робота на базе микроконтроллера **STM32F4** с поддержкой:
- 16 сервоприводов (две ноги, две руки) через **PCA9685**
- Двух моторов с ШИМ-управлением (L298N)
- Ультразвукового дальномера (HC-SR04)
- Гироскопа/акселерометра **MPU6050**
- Эмуляции джойстика через аналоговые входы

Проект написан в **Arduino IDE** для платформы **STM32F4** и является программной частью робота, которым можно управлять через веб-интерфейс (FastAPI) или десктопное приложение (PyQt5).

**Ключевые возможности:**
- Управление двумя дифференциальными моторами (вперёд/назад/повороты)
- Плавное управление 16 сервоприводами через I2C (PCA9685)
- Калибровка сервоприводов (4 конечности, 5 сервоприводов A–E)
- Чтение ультразвукового дальномера (расстояние в cm)
- Чтение данных с MPU6050 (акселерометр, гироскоп, температура)
- Режим эмуляции джойстика для тестирования без контроллера
- Готовые позы (руки вверх, ходьба, повороты головы)
- Асинхронное обновление сервоприводов (без задержек)

## Технологический стек

| Компонент       | Технология                      |
|-----------------|----------------------------------|
| MCU             | STM32F407 / STM32F405            |
| Среда разработки| Arduino IDE (STM32 Core)         |
| Драйвер моторов | L298N (GPIO + ШИМ)               |
| Сервоприводы    | PCA9685 (16 каналов, I2C)        |
| IMU             | MPU6050 (I2C)                    |
| Дальномер       | HC-SR04 (GPIO)                   |
| Джойстик        | Эмуляция через аналоговые пины   |
| Язык            | C++ (Arduino Framework)          |

## Быстрый старт

### 1. Установка Arduino IDE для STM32

```
# Установи Arduino IDE (если ещё нет)
sudo apt install arduino

# Добавь поддержку STM32 через менеджер плат:
# Файл → Настройки → Дополнительные ссылки для менеджера плат:
# https://github.com/stm32duino/BoardManagerFiles/raw/main/package_stmicroelectronics_index.json

# Затем: Инструменты → Плата → Менеджер плат → установи "STM32 Cores"
```

### 2. Установка библиотек

В Arduino IDE: **Скетч → Подключить библиотеку → Управлять библиотеками**

Установи:
- `Adafruit PWM Servo Driver Library` (для PCA9685)
- `MPU6050` (для гироскопа)

### 3. Сборка и загрузка

1. Открой `Izbushka_Firmware.ino`
2. Выбери плату: **Инструменты → Плата → STM32 Boards → Generic STM32F4 series → STM32F407VET6**
3. Выбери способ загрузки (ST-Link или Serial)
4. Нажми **Загрузить**
