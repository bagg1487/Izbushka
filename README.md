# Разработка конструктора "Избушка"

---

Основные компоненты:

1. **Десктопный сервер ([OpiServer](OpiServer))** - центральный узел управления на PyQt5 с голосовым интерфейсом, распознаванием лиц и жестов, воспроизведением мультимедиа и интеграцией с ИИ.
2. **Веб-сервер ([Face](Face))** - лёгкий веб-интерфейс на FastAPI для управления эмоциями, рисования жестами, игровым автоматом и диагностикой.
